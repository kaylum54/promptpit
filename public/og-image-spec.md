# OG Image Design Specification

## File Details
- **Filename:** `og-image.png`
- **Dimensions:** 1200 x 630 pixels
- **Format:** PNG (optimized for web)
- **Location:** `/public/og-image.png`

## Design Requirements

### Background
- **Style:** Dark gradient matching app theme
- **Colors:** Deep charcoal/slate gradient (e.g., #0f0f14 to #1a1a24)
- **Effect:** Subtle grid pattern or abstract neural network lines in the background
- **Opacity:** Background elements at 10-20% opacity to not distract from content

### Logo / Title
- **Text:** "PromptPit"
- **Position:** Centered horizontally, upper third of image
- **Font:** Bold, modern sans-serif (similar to Inter or similar to app font)
- **Size:** Large, prominent (approximately 80-100px)
- **Color:** White (#ffffff) or brand accent color
- **Effect:** Optional subtle glow or shadow for depth

### Tagline
- **Text:** "AI Model Debate Arena"
- **Position:** Centered, directly below the logo
- **Font:** Medium weight, same font family
- **Size:** Approximately 36-48px
- **Color:** Light gray (#a0a0a0) or muted accent color

### AI Model Icons/Names
- **Models to feature:**
  1. Claude (Anthropic)
  2. GPT-4o (OpenAI)
  3. Gemini (Google)
  4. Llama (Meta)
- **Layout:** Horizontal row, evenly spaced across the middle section
- **Style Options:**
  - Option A: Model name badges with colored backgrounds
  - Option B: Abstract icons representing each model
  - Option C: Small circular avatars with model initials
- **Colors:** Each model could have a distinct accent color:
  - Claude: Orange/Coral (#f97316)
  - GPT-4o: Green (#10b981)
  - Gemini: Blue (#3b82f6)
  - Llama: Purple (#8b5cf6)

### Debate Visual Representation
- **Concept:** Show the "versus" / debate nature of the platform
- **Options:**
  - Speech bubbles emanating from each model
  - VS symbols between models
  - Connecting lines with animated pulse effect (static representation)
  - Central "arena" or podium graphic
  - Trophy or gavel icon representing the AI referee
- **Position:** Integrated with the model icons section

### Additional Elements
- **Border:** Optional subtle border or rounded corners
- **Noise/Texture:** Light grain overlay for premium feel (2-5% opacity)
- **Safe Zone:** Keep important content within 1100x530 center area for platform cropping
